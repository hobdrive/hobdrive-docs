# Как перенести все настройки или сделать бэкап?

Все настройки хранятся в файлах:

```
user.events - события и заправки
user.gauge - пользовательские настройки отображения сенсоров
user.layout - пользовательские экраны (приложением не изменяется)
user.vehicles - список профилей авто пользователя
persistent.data - данные и показания накопительных датчиков
config.xml - главная конфигурация
track - папка с логами поездок
track_gen - сгенерированные поездки и записи
```

Для Windows версий файлы лежат в папке программы.

### Андроид версии до 1.6:

- на флеш диске в папке /sdcard (storage/emulated/0) / hobd/ (демо) или /sdcard (storage/emulated/0)/hobdrive (для полной версии).

### С Андроид версии 1.6.16 изменен путь:

- для полной версии  /storage/emulated/0/Android/data/hobdrive.android/files/hobdrive

- для демо версии на /storage/emulated/0/Android/data/hobdrivedemo.android/files/hobd

### Хобдрайв под iOS:

До этих файлов можно добраться через программу синхронизации iTunes, подключив устройство к компьютеру.

### Андроид версии 1.6.16 и выше

Системные файлы (не пользовательские) больше не доступны для прямого редактирования.

Это может усложнить "замену" картинок, раскладок, и так далее.

Для опытных пользователей с версии 1.6.29 можно пользоваться подпапокой "override".

При скрытом параметре "override-by-user" установленном в "true" - программа будет брать все замены из этой папки.

Таким образом можно переопределять любые системные ресурсы.

### Установка скинов

С версий после 1.6.29 все файлы скинов, картинки, нужно устанавливаться в подпапки с именем "override", "override-любойсуффикс".
